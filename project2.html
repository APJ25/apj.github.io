IGBT Switch Control Using Fuzzy Logic

Developed during my B.Tech at APJ Abdul Kalam Technological University, this project used fuzzy logic to dynamically control an IGBT switch and limit fault currents in power systems.

  • Designed fuzzy inference system to control duty cycle
  • Reduced harmonic distortion to under 7%
  • Simulated and validated model in MATLAB

Skills: Control logic design, fuzzy systems, simulation, IEEE publication
